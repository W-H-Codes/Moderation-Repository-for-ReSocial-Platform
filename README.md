# Moderation-Repository-for-ReSocial-Platform
This repository is contacted by the ReSocial platform for a list of banned user I.D.s and a hash of the current symmetric message encryption key!
If you want to report someone on the platform, send me an email with a screenshot of their message, and the computer generated message below it that shows their user I.D. code and the date it was sent. If the message goes of screen, please copy and paste the full text of both the message and the computer generated message below it into the email as well! Thanks!
